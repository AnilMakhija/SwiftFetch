# 📡 SwiftFetch

A lightweight Swift demo showcasing data fetching using URLSession — ideal for learning networking in iOS development with clean, testable code.

---

## 🚀 Overview

SwiftFetch demonstrates:

- ✅ Basic networking using URLSession  
- ✅ JSON decoding using Codable  
- ✅ Simple MVVM architecture for clarity and scalability  
- ✅ Swift best practices for async data loading

---

## 📦 Features

- Fetch data from a remote API  
- Decode JSON into Swift models  
- Display fetched data using SwiftUI (or UIKit if applicable)  
- Modular structure for reusability and testing

---

## 🛠 How to Use

### Clone the repo

```bash
git clone https://github.com/AnilMakhija/SwiftFetch.git
cd SwiftFetch
```

### Open in Xcode

- Open `SwiftFetch.xcodeproj` or `SwiftFetch.xcworkspace`
- Run the app in a simulator or on a device
